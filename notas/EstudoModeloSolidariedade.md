# Estudo do modelo de oferta do Solidariedade

- Tabela Pedido/Oferta - podemos chamar só de ordem:
    - ID oferta/pedido
    - ID produto
    - ID Pesquisador Proprietário
    - ID histórico status

- Tabela Produto:
    - ID produto
    - Quantidade
    - Tipo
    - Data de validade

- Tabela Pesquisador:
    - ID pesquisador
    - Instituição
    - Nome pesquisador
    - email

- Lista histórico de status:
    - ID histórico
    - Criado (Pedido) timestamp
    - Atendido timestamp
